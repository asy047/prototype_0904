what is web design

First, web design refers to all the designs that are shown through the web. All the designs shown through the web, such as display advertisement design, homepage design, UI design, and mobile advertisement design, are called web design. Therefore, UI designers can be called web designers, and shopping mall designers can also be called web designers.
